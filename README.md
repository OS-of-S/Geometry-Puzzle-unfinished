# Geometry Puzzle (ubfinished)
Пространственная головоломка, в которой игроку необходимо на каждом из уровней добыть синий объект. Для этого он может толкать и тянуть остальные объекты, но только за определённые грани.

Шейдер OUTLINE TOOLKIT by Charles Oliner: https://assetstore.unity.com/packages/tools/particles-effects/outline-toolkit-98020
Музыка: Всеволод Соколовский
Разработка: С. Смекты
